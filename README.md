# Было Тут — Android, Windows и Web

Публичная страница тестовых выпусков приложения «Было Тут». Основной сценарий — работа
на Android-смартфоне; также доступны самостоятельная Windows-версия и Web/PWA для браузера.
Рабочий репозиторий разработки остаётся закрытым.

## Что делает приложение

Android-приложение для поиска ближайших исторических фотографий рядом с современным фото, текущей геопозицией или вручную выбранной точкой. Приложение читает EXIF локально, ищет кандидатов по координатам, показывает результаты списком и на карте и позволяет сохранить выбранное сопоставление.

После выбора исторического фото можно включить камеру, подобрать современный ракурс и сделать снимок. Снимок сохраняется в `Pictures/Было Тут`, а карточка сопоставления — локально вместе со ссылкой на страницу PastVu, которой можно поделиться.

На карте результатов можно включить локальный слой «Мои фото»: координаты читаются на устройстве, а сами файлы никуда не отправляются.

Приложение не обещает точное совпадение ракурса: PastVu обычно хранит координаты объекта или места, а не точку фотографа, а сделанное фото может отличаться по расположению и характеристикам.

Использует открытый API PastVu. Не является официальным приложением PastVu. Права на
исторические изображения принадлежат их соответствующим правообладателям.

## Скачать и установить

1. Откройте [последний тестовый выпуск](https://github.com/klinnmy-ui/bylo-tut-download/releases/tag/v0.3.0-debug).
2. В разделе **Assets** скачайте `app-debug_bylo-tut.apk`.
3. Откройте загруженный APK на Android-смартфоне.
4. Если Android заблокирует установку, разрешите установку неизвестных приложений только для браузера или файлового менеджера, из которого открыт APK.
5. Нажмите «Установить», затем «Открыть».

Это тестовая debug-сборка, а не версия для магазина приложений.

## Запустить на Windows

1. Откройте [последний тестовый выпуск](https://github.com/klinnmy-ui/bylo-tut-download/releases/tag/v0.3.0-debug).
2. Скачайте `ByloTut-Windows-0.3.0-debug.zip` и полностью распакуйте архив.
3. Запустите `start-bylo-tut.bat` из распакованной папки.

Не запускайте отдельно извлечённый EXE: Windows-версии нужны соседние каталоги `app` и
`runtime`. Устанавливать Java отдельно для переносимого архива не требуется.

## Открыть в браузере

Универсальная web/PWA-версия доступна по адресу:
[https://klinnmy-ui.github.io/bylo-tut-download/](https://klinnmy-ui.github.io/bylo-tut-download/).

Она запускается на Android, iPhone и ПК без установки APK. Для камеры и текущей
геопозиции браузер запросит отдельное разрешение. Выбранная фотография читается
локально и не отправляется на сервер; для поиска PastVu передаются координаты и
параметры поиска. Возможности браузера зависят от устройства и могут отличаться
от Android-приложения.

В Web/PWA используется нейтральная картографическая подложка CARTO на данных OpenStreetMap
с резервным источником. Если подложка временно недоступна, маркеры, выбор точки и масштаб
остаются рабочими.

## Приватность

Современные фотографии не отправляются на внешние серверы. Для поиска PastVu передаются только координаты и параметры поиска. Разрешения на геопозицию, камеру и фотографии используются только соответствующими функциями приложения.
